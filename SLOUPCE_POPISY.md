# Sloupce v PVCHECK tabulce – přehled

Tabulka má dva řádky: názvy sloupců a jednoduchý popis.

---

| Sloupec | Popis |
|---------|-------|
| **source_file** | Ze kterého souboru data pocházejí |
| **Name** | Název typu měření (např. I-V Check) |
| **M1** | Číslo elektrárny nebo úroveň 1 |
| **M2** | Číslo řady panelů (string) |
| **M3** | Číslo modulu |
| **Num** | Pořadové číslo měření |
| **Type** | Kód typu záznamu |
| **Lenght** | Délka nebo počet vzorků |
| **DateTime** | Kdy se měřilo (datum a čas) |
| **text** | Krátký popis měření |
| **userText** | Poznámky od obsluhy |
| **outC** | Výstupní kód přístroje |
| **AHI** | Označení panelu (např. SPR-318E) |
| **Valtxt** | Typ testu (např. IV Test) |
| **ValVal** | Číselná hodnota |
| **ValMU** | Jednotka hodnoty |
| **Continuità** | Kontinuita obvodu (0 = OK) |
| **Continuità_unit** | Jednotka |
| **I Test** | Proud použitý při testu |
| **I Test_unit** | Jednotka proudu (mA) |
| **Informazioni generali** | Obecné informace |
| **Informazioni generali_unit** | Jednotka |
| **Irraggiamento** | Kolik světla dopadá na panel (W/m²) |
| **Irraggiamento_unit** | Jednotka |
| **Irraggiamento minimo** | Minimální osvit pro měření |
| **Irraggiamento minimo_unit** | Jednotka |
| **Isc avg** | Průměrný zkratový proud |
| **Isc avg_unit** | Jednotka (A) |
| **Isc opc** | Zkratový proud při provozní teplotě |
| **Isc opc_unit** | Jednotka (A) |
| **Isc stc** | Zkratový proud při standardních podmínkách |
| **Isc stc_unit** | Jednotka (A) |
| **Isolamento** | Stav izolace |
| **Isolamento_unit** | Jednotka |
| **Modalità temperatura** | Režim měření teploty (nastavení) |
| **Modalità temperatura_unit** | Jednotka |
| **Numero Moduli** | Kolik panelů se měřilo |
| **Numero Moduli_unit** | Jednotka |
| **PVCHECK S/N** | Sériové číslo měřicího přístroje |
| **PVCHECK S/N_unit** | Jednotka |
| **R Lim** | Minimální povolený odpor |
| **R Lim_unit** | Jednotka |
| **RI** | Izolační odpor (čím vyšší tím lépe) |
| **RI_unit** | Jednotka (MΩ) |
| **RPE** | Odpor uzemnění (čím nižší tím lépe) |
| **RPE_unit** | Jednotka (Ω) |
| **Ri** | Odpor |
| **Ri_unit** | Jednotka |
| **Ri +** | Izolační odpor z plusu |
| **Ri +_unit** | Jednotka |
| **Ri -** | Izolační odpor z minusu |
| **Ri -_unit** | Jednotka |
| **Rlim** | Limit odporu pro vyhodnocení |
| **Rlim_unit** | Jednotka |
| **Rmax** | Maximální naměřený odpor |
| **Rmax_unit** | Jednotka |
| **Temperatura** | Teplota panelu nebo okolí |
| **Temperatura_unit** | Jednotka (°C) |
| **Tol I** | Povolená odchylka proudu v procentech |
| **Tol I_unit** | Jednotka |
| **Tol V** | Povolená odchylka napětí v procentech |
| **Tol V_unit** | Jednotka |
| **Unità remota** | Vzdálená jednotka (0 = nepoužita) |
| **Unità remota_unit** | Jednotka |
| **V0 nom** | Nominální napětí |
| **V0 nom_unit** | Jednotka |
| **VEN** | Napětí z minusu |
| **VEN_unit** | Jednotka |
| **VEP** | Napětí z plusu |
| **VEP_unit** | Jednotka |
| **VPN** | Součet absolutních napětí plus a minus |
| **VPN_unit** | Jednotka |
| **Vmis (-)** | Měřené napětí minus |
| **Vmis (-)_unit** | Jednotka |
| **Vnom** | Nominální napětí řetězce |
| **Vnom_unit** | Jednotka (V) |
| **Voc avg** | Průměrné napětí naprázdno |
| **Voc avg_unit** | Jednotka (V) |
| **Voc opc** | Napětí naprázdno při provozní teplotě |
| **Voc opc_unit** | Jednotka (V) |
| **Voc stc** | Napětí naprázdno při standardních podmínkách |
| **Voc stc_unit** | Jednotka (V) |
| **Vtest** | Napětí při testu izolace |
| **Vtest_unit** | Jednotka (V) |
| **alpha** | Jak se mění napětí s teplotou |
| **alpha_unit** | Jednotka (%/°C) |
| **beta** | Jak se mění proud s teplotou |
| **beta_unit** | Jednotka (%/°C) |
